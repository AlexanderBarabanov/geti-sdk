class sc_api_tools.deployment.data_models.ROI(labels: List[sc_api_tools.data_models.label.ScoredLabel], modified: Optional[Union[str, datetime.datetime]] = None, id: Optional[str] = None, labels_to_revisit: Optional[List[str]] = None, *, shape: sc_api_tools.data_models.shapes.Rectangle, original_shape: sc_api_tools.data_models.shapes.Shape)

Bases: sc_api_tools.data_models.annotations.Annotation

This class represents a region of interest for a given image. ROIs are generated for intermediate tasks in the pipeline of a project, if those tasks produce local labels (for instance a detection or segmentation task).
