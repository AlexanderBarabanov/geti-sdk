geti_sdk.data_models

Introduction

The data_models package contains the SDK representation for all entities in GETi, such as AnnotationScene, Image, Project and Model and many more.

When interacting with the GETi cluster through the geti_sdk.sc_rest_client.Geti or the rest_clients, all entities retrieved from the cluster will be deserialized into the data models defined in this package.
